## 1. 문서 목적

Phase 0(MVP)은 공개 데이터셋 메타데이터 기반의 분류+추천 데모로 끝난다. Phase 1은 실제 음원 스트리밍 서비스(예: Spotify Web API)와 연동해 추천 결과를 실제로 재생/검색할 수 있게 하는 확장을 다룬다.

- **다루는 범위**: 외부 스트리밍 서비스 API 연동 방식, 인증, 데모 데이터셋의 곡과 스트리밍 서비스 카탈로그 간 매칭 전략.
- **다루지 않는 범위**: Phase 0의 모델 학습·분류·추천 로직 자체(변경 없음, `docs/prd.md` 참조).
- **이 문서로 내릴 결정**: 연동 대상 서비스 선정, 인증 방식, 매칭 실패 시 처리.

## 2. 프로젝트 개요

이 phase는 아직 범위·가정·리스크 정의 단계의 스텁이다. Phase 0 MVP가 완료되고 실제로 이 확장을 추진하기로 결정된 뒤에 본문을 채운다. 구현은 본 문서가 채워지고 승인된 뒤에만 시작한다(워크스페이스 Phase 분리 규칙).

## 24. 열린 질문

- 연동 대상 서비스(Spotify Web API 등)의 약관과 무료 티어 제한.

- 이 phase를 실제로 추진할지 여부 자체가 아직 결정되지 않았다.
